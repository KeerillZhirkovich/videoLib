++++++++++++++++++++
Сборка проекта:

Для успешной сборки проекта должен быть установлен и правильно настроен Maven.


- В дериктории с проектом кликнуть правой кнопой мыши при зажатой клавише Shift.
- В выпадающем меню выбрать пункт "Открыть окно команд" (Командная строка, CMD here или аналогичное)
- В появившейся командной строке прописать: mvn clean install
- При выполнении всех условий проект будет собран в target.


++++++++++++++++++++
Запуск проекта:

Для успешного запуск проекта должен быть установлен и правильно настроен JDK.

- В папке target найти файл information_system-1.0-SNAPSHOT-jar-with-dependencies.jar
- Открыть найденный файл двойным кликом по нему. Проект запустится автоматически.

- При возникновени проблем с запуском по двойному клику необходимо создать рядом с файлом проекта 
файл *.bat, в котором прописать:

@ECHO OFF
start javaw.exe -jar information_system-1.0-SNAPSHOT-jar-with-dependencies.jar
